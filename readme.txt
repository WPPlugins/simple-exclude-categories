Hide posts in categories from WordPress Homepage

== Description ==
English: Plugin help hide all posts in one or more category on Homepage. Go to \"Settings» Reading \"to choose the category need to hide

Vietnamese: Plugin giúp ẩn bài posts trong một hoặc nhiều category trên trang Homepage. Vào \"Settings » Reading\" để chọn category cần ẩn

== Installation ==
1. Download zip file
2. Upload the Simple Exclude Categories plugin to your blog
3. Activate it
